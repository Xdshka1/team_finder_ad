# TeamFinder

TeamFinder — веб-приложение на Django для поиска единомышленников и участников в pet-проекты.

Пользователи могут регистрироваться, создавать проекты, редактировать профиль, добавлять проекты в избранное и присоединяться к проектам других пользователей.

Проект выполнен по варианту 1:  
«Избранное» и фильтрация пользователей.

## Реализованная функциональность

- регистрация пользователей;
- вход и выход из аккаунта;
- смена пароля;
- публичная страница пользователя;
- редактирование профиля;
- список зарегистрированных пользователей;
- создание проекта;
- редактирование своего проекта;
- просмотр страницы проекта;
- завершение своего проекта;
- присоединение к чужому проекту;
- отказ от участия в проекте;
- добавление проекта в избранное;
- удаление проекта из избранного;
- страница избранных проектов;
- фильтрация пользователей по критериям варианта 1;
- админ-панель Django.

## Примечание

Проект выполнен по варианту 1: реализованы избранные проекты и фильтрация пользователей.

PostgreSQL внутри контейнера работает на стандартном порту `5432`, а на локальную машину проброшен через порт `5436`.

Если порт `5436` занят, его можно заменить в двух местах:

yaml
# docker-compose.yml
ports:
  - "5436:5432"

и:

env
# .env
POSTGRES_PORT=5436
